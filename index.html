The KochChart Density Altitude Software Tool

Introduction

Welcome! This site is your source for the KochChart educational software tool. This tool compares various estimating methods (including, but not limited to, the Koch Chart) for assessing the impact of density altitude on take-off and rate-of-climb performance. In addition, the tool can estimate the effects of wind, runway slope and runway surface on take-off distance.

This is a free educational tool created to raise awareness about the significant effect density altitude has on aircraft performance, and is especially important for pilots flying at high-elevation airports.

Why the name 'Koch Chart'? This program started as a relatively simple digital version of the venerable Koch Chart nomograph (the roots of which are somewhat obscure). The user would type in key parameters to produce the chart. But eventually automated METAR collection and others' estimating methods were included. Now the program does much more than produce a Koch Chart, but the name lives on!

The algorithms used in the software were originally developed for light, single-engine, normally aspirated (i.e., not-turbocharged) piston aircraft. They are not applicable to other types of aircraft.
